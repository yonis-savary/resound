# Resound

Listen to your CD/MP3 library remotely !

## Features

- Music Player
- Library Tag Parsing
- Spotify info fetching

## Installation

Requirements: docker

```bash
# Setup
git clone git@github.com:yonis-savary/resound-v2-nuxt.git
cd resound-v2-nuxt
docker compose up -d
```
